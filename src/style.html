Position

The position property in CSS is used to specify the positioning method of an element. It allows you to control how an element is positioned on the page and how it interacts with other elements. There are several values for the position property, including static, relative, absolute, fixed, and sticky.

Static Position

Elements with a static position are positioned according to the normal flow of the document. They are not affected by the top, bottom, left, or right properties.

div with static position

Fixed Position

Elements with a fixed position are positioned relative to the viewport, which means they stay in the same place even when the page is scrolled. You can use the top, bottom, left, and right properties to specify the exact position of the element.

div with fixed position

Absolute Position

Absolute position elements are similar to fixed position elements. However, fixed elements are fixed to the browser window, while absolute elements are abosulte to the page. This means that absolute elements will scroll with the page, while fixed elements will remain in the same position on the screen regardless of scrolling. Absolute elements are positioned relative to their nearest positioned ancestor (an element with a position other than static), or if there is no such ancestor, they are positioned relative to the initial containing block (the html element).

Placing an absolute element inside a relative element allows you to position the absolute element relative to the boundaries of the relative element. This is a common technique for creating complex layouts and controlling the positioning of elements on a web page. For example, if you have a div with position: relative and an absolute element inside it, you can use the top, bottom, left, and right properties to position the absolute element within the boundaries of the relative div.

div with absolute position

Relative Position

Relative position elements are positioned relative to their normal position in the document flow. You can use the top, bottom, left, and right properties to specify how much the element should be offset from its normal position. When you set an element's position to relative, it remains in the normal document flow, and other elements will still be positioned as if the relatively positioned element were in its original position. When you place an absolute element inside a relative element, the absolute element is positioned relative to the boundaries of the relative element. This allows you to control the positioning of the absolute element within the context of the relative element, which can be useful for creating complex layouts and ensuring that elements are positioned correctly on the page.

Z-Index

By default, elements are stacked in the order they appear in the HTML document, with later elements appearing on top of earlier ones. However, by using the z-index property, you can change this stacking order and specify which elements should appear in front of or behind others. Z-index is a CSS property that controls the stacking order of elements on a web page. It determines which elements appear in front of or behind other elements when they overlap. The z-index property only works on positioned elements (those with a position value of relative, absolute, fixed, or sticky). For example, a div with a z-index of 2 will appear in front of a div with a z-index of 1, and a div with a z-index of 0 will appear behind both of them.
