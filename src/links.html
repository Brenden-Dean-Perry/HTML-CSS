Links (<a>)

Links are created using the <a> tag. The href attribute specifies the URL of the page the link goes to.

Absolute Links (References)

Absolute (external) links specify the full URL of the page including external websites.

Relative Links (References)

Relative (internal) links specify a path relative to the current page:

Download Link

You can download the example file by clicking the link below. It adds a download attribute to the anchor tag, which prompts the browser to download the file instead of navigating to it.
Note: downloads may not work in all browsers some may simply navigate to the file instead of downloading it.
You may also specify a different filename for the downloaded file by using the download=new_file_name.ext attribute with a value.

Download Example Image

Email Link

You can create an email link using the mailto: scheme in the href attribute.

Note in practice, this is not advisable for public websites due to spam risks as crawlers can easily harvest email addresses.

Send an Email

Phone number Link

You can create a phone number link using the tel: scheme in the href attribute.

Note in practice, this is not advisable for public websites due to spam risks as crawlers can easily harvest phone numbers.
